# love-plus [![Build Status](https://travis-ci.org/rameshvarun/love-plus.svg)](https://travis-ci.org/rameshvarun/love-plus)
Utilities for writing Love2D games in Atom.

## Features
- Automatically installs the following:
  - tool-bar
  - language-lua
  - linter
  - linter-luacheck
  - hyperclick
- Autocomplete
